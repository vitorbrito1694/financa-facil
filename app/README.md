# Finança Fácil - App (Frontend)

Interface do usuário para o sistema Finança Fácil, desenvolvida com **Next.js**.

## Tecnologias

- **Framework**: Next.js 14
- **Estilização**: TailwindCSS
- **Linguagem**: TypeScript

## Configuração

### 1. Instalação

Entre na pasta `app` e instale as dependências:

```bash
cd app
npm install
```

### 2. Executando o Projeto

Para rodar o servidor de desenvolvimento:

```bash
npm run dev
```

> **Nota**: O projeto está configurado para rodar na porta **1240**.
> Acesse: [http://localhost:1240](http://localhost:1240)

## Estrutura

- `src/app`: Páginas e rotas do Next.js (App Router).
- `src/components`: Componentes reutilizáveis.
- `src/styles`: Estilos globais.

## Variáveis de Ambiente

Se necessário, crie um arquivo `.env.local` na raiz de `app`.
